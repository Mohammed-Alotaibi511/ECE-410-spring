# Mohammed Alotaibi  
 ECE 410 Spring 2026 – CNN Accelerator Project

This repository contains all milestones and final deliverables for the ECE 410 hardware accelerator project. The project implements a CNN style compute accelerator and evaluates its performance, area, timing, power, and arithmetic intensity using simulation and OpenLane synthesis.
