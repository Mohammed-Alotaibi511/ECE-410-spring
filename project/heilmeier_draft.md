# Heilmeier Questions
 # 1. What are you trying to do?
I want to design and analyze a neural network model that performs image classification using a simple architecture. The goal of this is to know how computation and memory and performance trade-offs affect machine learning models.

 # 2. How is it done today, and what are the limits?
 Deep learning models like ResNet are used for image classification. They are accurate but need high computation and memory, making them expensive to run on hardware. The main drawback is inefficiency in resource usage.

 # 3. What is new in your approach?
My approach is focusing on analyzing and understanding the computational workload of neural networks. Instead of just using models I want to break down MAC operations and memory usage and arithmetic intensity to understand performances better. This can help in designing more efficient systems for AI workloads.
