\# M4 README



This folder contains the final M4 submission for the CNN convolution accelerator project.



\## Files



\- `rtl/top.sv` — Final integrated top module.

\- `rtl/compute\_core.sv` — Final compute core RTL.

\- `rtl/interface.sv` — Final interface RTL.

\- `tb/tb\_top.sv` — Final end-to-end testbench.

\- `sim/final\_run.log` — Final simulation log showing PASS.

\- `sim/final\_waveform.png` — Final waveform image.

\- `synth/config.json` — OpenLane 2 synthesis configuration.

\- `synth/openlane\_run.log` — Full OpenLane run log.

\- `synth/timing\_report.txt` — Timing report with slack and critical path.

\- `synth/area\_report.txt` — Area report and cell counts.

\- `synth/power\_report.txt` — Power estimate report.

\- `bench/benchmark.md` — Hardware vs software benchmark summary.

\- `bench/benchmark\_data.csv` — Raw benchmark/projection data.

\- `bench/roofline\_final.png` — Final roofline plot.

\- `report/design\_justification.pdf` — Final design justification report.

\- `report/figures/final\_waveform.png` — Waveform figure used in report.

\- `report/figures/roofline\_final.png` — Roofline figure used in report.
